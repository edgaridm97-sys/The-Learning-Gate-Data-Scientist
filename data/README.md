# The Learning Gate – Data Scientist (Módulo 1)

Este repositorio contiene los datos y notebooks del **Reto del módulo 1: Visualización de datos con Python**.

## Cómo reproducir (Colab)
1. Abre el notebook en Google Colab.
2. Presiona **Ejecutar todo**.
3. El notebook descargará los datos automáticamente desde GitHub.
